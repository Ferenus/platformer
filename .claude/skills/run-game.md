# Run Game Skill

Use this skill to launch the platformer game for the user.

## Trigger
When the user says "run game", "start game", "play game", or "launch game".

## Steps

1. Activate the virtual environment: `source venv/bin/activate`
2. Run: `python game.py`
3. The game window will open. Controls:
   - Arrow keys or WASD: Move and jump
   - SPACEBAR: Shoot
   - Arrow UP / W at open door: Enter door to complete level
   - R: Restart (after win/death)
   - ESC: Quit
